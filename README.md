MyAnime

MyAnime is a simple anime recommendation website built using HTML, CSS, and basic JavaScript. It provides a home page, a recommendations page, and individual genre and anime pages.

Features

Navigation bar linking Home and Recommendations pages

Recommendation section with genres like Action, Adventure, etc.

Genre subpages displaying related anime with poster-style layouts

Individual anime pages with details like title, type, duration, and description

Responsive and minimal design consistent across all pages


Structure

index.html – Home page

recommendations.html – Main recommendations page

/recomms/animes/ – Contains individual anime pages

/recomms – Contains genre subpages

style.css – Main stylesheet for all pages


Setup

1. Clone or download the project folder.


2. Open index.html in a browser to start.


3. Use the navbar to navigate between pages.
